# Universities_Scraping

A Python web scraping project that extracts university and course information from the IDP website and exports structured data into a formatted Excel file.

Project Overview :
Scrapes top 10 universities.
Extracts course details for each university.
Handles missing data safely.
Exports structured data into an Excel file.
Dynamic scraping.

 Technologies used :
 Python
 BeautifulSoup4
 pandas
